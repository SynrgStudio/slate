# ADR-0005: Lightweight Markdown preview as optional mode

Status: proposed
Date: 2026-05-24
Supersedes: none
Superseded by: none

## Context

Slate supports simple text editing with optional lightweight Markdown preview toggled by Ctrl+M.

Evidence: README.md

## Decision

Keep Markdown preview simple and optional rather than making Slate a full Markdown editor. This ADR is proposed and must be reviewed before acceptance.

## Consequences

Reduces complexity while supporting common note-taking use cases.
