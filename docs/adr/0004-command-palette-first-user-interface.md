# ADR-0004: Command-palette-first user interface

Status: proposed
Date: 2026-05-24
Supersedes: none
Superseded by: none

## Context

The README describes Slate as command palette driven with no menubar and no toolbar.

Evidence: README.md

## Decision

Expose actions through keyboard shortcuts and Ctrl+P command palette rather than traditional menus/toolbars. This ADR is proposed and must be reviewed before acceptance.

## Consequences

The UI remains minimal, but discoverability depends on shortcut documentation and palette quality.
